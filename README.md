# Exercicio pratico PROJOO <br>
Sistema em C# que gerencia reservas de salas para estudantes e professores, implementando os padrões Strategy (políticas FIFO e Prioridade Docente), Singleton, Factory Method e Observer. Para usar, basta compilar e executar o projeto no Visual Studio – o programa demonstra automaticamente os cenários de reserva, conflitos, cancelamentos por prioridade e geração de relatório diário.
